# Document QA - High-Performance AI Interface

The official frontend for the Document QA platform. This is a modern, responsive **React** application built to provide a seamless auditing experience for complex documents through an intelligent RAG-powered chat interface.

## 🚀 Key Features

* **Real-time Auditing Interface:** A clean, "Command & Navigate" style UI designed for high-density document Q&A.
* **Session-Aware Chat:** Integrated session ID management that syncs with the FastAPI backend for persistent, context-aware conversations.
* **Smart UX Components:** Includes auto-scrolling chat windows, markdown rendering for AI responses, and dynamic file upload status.
* **Responsive Architecture:** Fully optimized for both desktop auditing and mobile quick-checks using Tailwind CSS.
* **Industrial Standard:** Built with modular components following the Atomic Design pattern for easy scalability.

## 🛠️ Tech Stack

* **Framework:** React.js / Next.js
* **Styling:** Tailwind CSS
* **State Management:** React Hooks & Context API
* **Icons:** Lucide-React
* **Deployment:** Docker & Vercel-ready
